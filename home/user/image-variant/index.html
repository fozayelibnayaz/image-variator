How It Works — 100% Pure Camera Angles Only (Zero Filters)

  • Number of variants: Set 1 to 24. You get that many independent variants.
  • Manual camera angle assignment: Every variant has its own dropdown. You can choose any of the 24 camera takes (Standard Frontal, Low Angle, Macro, Dutch Tilt, Overhead, Fisheye-style, Vertical Portrait, Texture Detail, etc.). You control every single one.
  • Randomize button: One click gives a fresh set of unique camera angles.
  • 100% pure geometry — NO FILTERS AT ALL:
    • No CSS filters, no color/contrast/saturation/hue/sepia/brightness changes.
    • No lighting effects, no vignettes, no dramatic overlays, no blur.
    • Only real camera work: different source crops (zoom in/out), different framing offsets, and real canvas rotation for angle looks.
    • Result: each variant looks new because the camera position, distance, and tilt are different — exactly like shooting the same object with different camera setups.
  • Visual Concept: Automatically created from the camera angle you chose (e.g. "Low Angle Perspective — dramatic low angle, strong vertical emphasis"). This drives the French SEO.
  • French SEO (content-aware + direct mapping): Title, Alt, Caption, Description are complete grammatically correct French sentences that are now directly matched to the exact camera angle/framing you assigned for that variant (perfect relevance). 100% uniqueness guaranteed across the batch.
  • Per-variant controls: 🔎 Fetch or ✨ Generate on any card or table row re-creates SEO for just that variant using its camera concept (while keeping global uniqueness). ⬇️ Download saves the exact rendered image.
  • Upload & formats: Any number of base images (PNG/JPG/WEBP). Choose output format (JPG/PNG/WEBP).
  • Exports: ZIP contains all rendered images (in chosen format) + metadata.txt (with your chosen camera angles + concepts) + CSV. Separate CSV also available.

Double-click index.html to run locally. 100% client-side. No server or internet required.
