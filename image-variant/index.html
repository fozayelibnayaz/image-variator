How It Works (English)

  • Upload: Drop or select up to 4 base images (one per page). If none uploaded, realistic placeholders are used automatically.
  • Generate Variants: Creates 4 completely unique variants. Visuals simulated with real canvas filters + framing based on assigned camera/light. Every variant uses a different (camera take + light profile) pair.
  • French SEO: All Alt Text, Title, Caption, Description are full grammatically correct French sentences. Strict 100% uniqueness: no sentence or phrase is reused across any of the 16 fields in the batch.
  • Per-variant controls: Use 🔎 Fetch or ✨ Generate on any card to refresh only that variant’s SEO while automatically avoiding phrases already used by the other three variants.
  • Editing: Change text in the table anytime. Character counters turn red if over limit (but generation always produces complete sentences under limits).
  • Exports: ZIP includes the 4 processed unique JPG variants + full metadata text file + CSV. CSV export is also available separately.
  • Guarantees: 24 camera takes × 5 light profiles = 120 possible pairs. Shuffled master list ensures the 4 variants always get distinct pairs. Text uniqueness enforced with global phrase tracking + retries.
